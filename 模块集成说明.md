# 命令集管理功能集成指南

## 概述

本指南说明如何将命令集管理功能集成到主程序 `complete_command_sender.py` 中。

## 模块文件说明

在 `modules` 目录下已创建以下文件：

| 文件名 | 功能 |
|--------|------|
| `__init__.py` | 模块初始化文件 |
| `command_set_manager.py` | 命令集管理器（CRUD操作） |
| `desktop_export.py` | 桌面导出管理器（快捷方式创建） |
| `command_set_integration.py` | 集成辅助类（简化集成过程） |

## 快速集成步骤

### 步骤 1: 在主程序中添加导入语句

在 `complete_command_sender.py` 文件顶部（其他导入语句附近）添加：

```python
# 命令集管理模块
try:
    from modules.command_set_integration import CommandSetIntegration
    COMMAND_SET_AVAILABLE = True
except ImportError as e:
    COMMAND_SET_AVAILABLE = False
    print(f"命令集模块导入失败: {e}")
```

### 步骤 2: 在 `CommandSenderApp` 类中添加初始化

在 `__init__` 方法中添加（约在配置加载之后）：

```python
# 初始化命令集管理集成
self.command_set_integration = None
if COMMAND_SET_AVAILABLE:
    try:
        self.command_set_integration = CommandSetIntegration(self)
    except Exception as e:
        print(f"命令集集成初始化失败: {e}")
```

### 步骤 3: 在 `create_menu` 方法中添加菜单

在工具菜单区域添加命令集子菜单（约第1467行附近）：

```python
# 在 tools_menu.add_separator() 和 tools_menu.add_command(label="设置") 之间添加：

# 命令集子菜单
if self.command_set_integration:
    commandset_menu = tk.Menu(tools_menu, tearoff=0)
    tools_menu.add_cascade(label="命令集管理", menu=commandset_menu)
    commandset_menu.add_command(label="保存选区为命令集", 
                                command=self.command_set_integration.save_selection_as_command_set)
    commandset_menu.add_command(label="管理命令集", 
                                command=self.command_set_integration.manage_command_sets)
    commandset_menu.add_separator()
    commandset_menu.add_command(label="导出到桌面", 
                                command=self.command_set_integration.export_command_set_to_desktop)
    tools_menu.add_separator()
```

### 步骤 4: 添加工具栏按钮（可选）

在 `create_toolbar` 方法末尾添加：

```python
# 命令集按钮
if hasattr(self, 'command_set_integration') and self.command_set_integration:
    ttk.Separator(toolbar, orient=tk.VERTICAL).pack(side=tk.LEFT, fill=tk.Y, padx=5)
    ttk.Button(toolbar, text="命令集", 
              command=self.command_set_integration.manage_command_sets).pack(side=tk.LEFT, padx=2)
```

### 步骤 5: 添加快捷键绑定（可选）

在 `__init__` 方法的事件绑定区域添加：

```python
# 命令集相关快捷键
if hasattr(self, 'command_set_integration') and self.command_set_integration:
    self.root.bind('<Control-Shift-S>', 
                  lambda e: self.command_set_integration.save_selection_as_command_set())
```

## 使用说明

### 创建命令集

1. 在文本编辑器中选择多行命令
2. 点击菜单 `工具` → `命令集管理` → `保存选区为命令集`
3. 输入命令集名称和描述
4. 点击"保存"

### 使用命令集

1. 点击菜单 `工具` → `命令集管理` → `管理命令集`
2. 在列表中选择要执行的命令集
3. 点击"执行"按钮

### 导出到桌面

1. 点击菜单 `工具` → `命令集管理` → `导出到桌面`
2. 选择要导出的命令集
3. 设置发送延迟
4. 点击"导出"

## 常见问题

### Q: 导入模块失败怎么办？

A: 检查以下几点：
1. 确保 `modules` 目录与主程序在同一目录下
2. 确保 Python 可以访问该目录
3. 检查是否有语法错误

### Q: 导出快捷方式失败怎么办？

A: 系统会尝试创建批处理文件作为备选方案。如果 win32com 不可用，快捷方式功能将受限。

### Q: 如何卸载命令集功能？

A: 
1. 删除 `modules` 目录
2. 移除添加的导入语句
3. 移除菜单和工具栏添加的代码
